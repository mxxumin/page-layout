> `“瀑布流布局”`是一种常见的用于展示图片、文字的布局方式，它表现为宽度相同，高度不同，错落但又规整的感觉，并且在滚动页面时，可以不断的加载新的内容。

### 瀑布流布局特点：
**1、内容的每一项宽度相同，高度不同；**

**2、滚动页面时，可以不断的加载新的内容；**
