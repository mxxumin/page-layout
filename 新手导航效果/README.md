> `“新手导航效果”`是引导用户一步步熟悉使用网站的一种效果，可以让用户更方便地了解网站。

### 新手导航效果特点：

**1、通过cookie判断用户是否已经访问过，如果是初次访问则显示导航；**

**2、用户可以按照引导一步步了解网站特点，也可以随时关闭导航；**
