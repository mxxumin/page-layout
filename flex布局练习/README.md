## 2、圣杯布局

> 圣杯布局是一种最常见的网站布局，页面从上到下，分为三个部分：头部（header）、中间容器（container）、尾部（footer）。其中中间容器从左到右又分为：导航（nav）、主栏（main）、侧栏（aside）。这种布局特点是：在中间容器（container）渲染时，主栏（main）先渲染，导航（nav）和侧栏（aside）宽度固定，主栏（main）宽度自适应。
