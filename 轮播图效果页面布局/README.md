> 在看过支付宝网站个人用户页面的时候，觉得它的轮播图效果的页面展示非常棒，给人一种新的体验，于是就模仿出了一个精精精精简版的，我给它起名叫`“轮播图效果的网页布局”`。

### 轮播图效果的网页布局特点：
**1、整个页面由多个宽高等于浏览器宽高的模块组成；**

**2、点击向下箭头或者"进度条"（侧边圆圈组成的）时会进行模块之间的切换；**

**3、当显示到最后一个模块的时候，箭头会消失；**
